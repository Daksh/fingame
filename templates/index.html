About

FinGame

We want to simplify banking for children. We will implement a gamified interface for the users using CORDA api and enjoy learning banking. The game seeks to simplify transactions and step by step leverage an increasing difficulty in learning the basics of banking, ledgers and loan system. We planned to build a simpler UI for Corda and a pre-configured set of values for banks and places, so that users can run the simulations directly and start learning about banking from absolutely simple basics.
